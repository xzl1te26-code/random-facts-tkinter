# Интересные факты

Простая программа на Tkinter, которая показывает случайные интересные факты.

## Как это работает

Запускаешь программу — открывается окошко. Нажимаешь на зелёную кнопку — появляется какой-нибудь факт. Всё просто.

## Что внутри

- Всего 5 фактов (бананы радиоактивные, киты много едят, пчёлы запоминают лица и т.д.)
- Кнопка и текст с переносом по строкам
- Окошко 400x250 пикселей

## Как запустить

```bash
python main.py
